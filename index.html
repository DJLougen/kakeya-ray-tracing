How It Works

The Kakeya conjecture studies line segments in different directions. In ray tracing, each ray is a line segment. Rays hitting reflective surfaces (metal, glass) require 5-10 bounces, while sky rays need only 1 bounce.

Traditional uniform tiling creates tiles with wildly different workloads. Polynomial partitioning divides the image using workload-weighted medians, ensuring each GPU warp receives balanced work, not balanced pixel counts.

The result: 1.5-1.6× speedup with partition overhead of only 3-54ms (negligible compared to render times of 0.4-6.9s).
